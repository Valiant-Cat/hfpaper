## 📄 Attend Before Attention: Efficient and Scalable Video Understanding via Autoregressive Gazing

- **链接**: https://huggingface.co/papers/2603.12254
- **阅读来源**: HTML

### 1. 应用领域
多模态大模型（MLLM） / 计算机视觉 - 高效长视频与高分辨率视频理解。

### 2. 一句话核心贡献
提出了一种仅3M参数的前置轻量级自回归模块 **AutoGaze**，在视频输入视觉编码器（ViT）之前动态剔除时空冗余图块，突破了传统模型处理所有像素的算力瓶颈，实现了多模态大模型对 1000帧、4K分辨率长视频的高效处理，并构建了首个超清长视频问答基准 HLVid。

### 3. 使用指南
*   **输入**：任意时长、任意分辨率的原始视频帧序列。
*   **输出**：过滤去重后的最小化多尺度图像块（Patches）索引序列，仅包含具备运动或新信息的关键区域。
*   **使用流程**：将 AutoGaze 作为“前置过滤器”插入在标准 ViT 或 MLLM 之前。它会逐帧评估并挑选图块，实时预测重构损失，一旦预测损失低于用户设定的阈值（如0.7），即自动停止该帧的图块采样，随后将选出的图块输入给微调后的多尺度 ViT 进行常规处理。
*   **硬件与部署**：极其轻量（仅包含一层卷积编码器和四层 Transformer 解码器），不仅无需特殊昂贵硬件，反而能大幅降低原大模型处理长视频时的显存占用（避免 OOM）。
*   **开源情况**：论文中提及了 Project page 占位符，且核心代码基于开源架构（如 LLaMA 3 解码器、NVILA 等），通常意味着相关代码和基准集将被开源。

### 4. 主要创新点
1. **前置冗余消除（Attend Before Attention）**：有别于以往在 ViT 内部或 LLM 层级进行 Token 剪枝/聚合的方法，AutoGaze 在像素图块进入 ViT 之前就进行了硬提取（Hard Selection），彻底消除了视觉编码阶段处理无用静态背景的巨大计算浪费。
2. **多尺度自回归注视与动态停止**：模型以类似语言模型生成 Token 的自回归方式挑选图块，并支持多尺度（例如对高频细节使用细粒度，纯色背景使用粗粒度）。同时引入了一个线性头来实时预测视频重构损失，实现了基于质量要求的自适应提前停止（Early Stopping）。
3. **引入 LLM 范式的两阶段训练法**：
   * *预训练*：基于贪婪搜索生成的次优注视序列，进行下一词预测（NTP）的交叉熵训练。
   * *强化学习后训练*：使用基于重构奖励的简化版同策略强化学习算法（GRPO）探索更优的采样序列，使其学会忽略重复信息并精准捕捉高动态/高细节区域。

### 5. 实验效果
*   **极致的加速与压缩**：在维持下游任务性能无明显下降的前提下，可将视频处理所需的图像块数量成倍减少（例如处理 30-FPS 4K 视频时只需 1% 的图块），使 ViT 和 MLLM 的推理速度**最高提升 19 倍**。
*   **突破显存瓶颈（Scalability）**：在传统基线模型处理超过 256 帧就耗尽显存的情况下，AutoGaze 成功将 NVILA-8B 模型无缝扩展至处理 **1000帧、4K分辨率** 的长视频。
*   **核心数据集表现**：
    * 在通用视频基准 VideoMME 上取得了 **67.0%** 的优异成绩。
    * 在作者首创的高分辨率长视频问答基准（**HLVid**，包含长达5分钟的4K视频）上，加入 AutoGaze 扩展后的模型准确率从 42.5% 大幅跃升至 **52.6%（提升10.1%）**，且超越了此前的 SOTA 极强开源模型（如 Qwen2.5-VL-7B） **4.5%**。

## 📄 SpecEyes: Accelerating Agentic Multimodal LLMs via Speculative Perception and Planning

- **链接**: https://huggingface.co/papers/2603.23483
- **阅读来源**: HTML

### 1. 应用领域
多模态大语言模型 (MLLMs) 推理优化、智能体架构 (Agentic AI) 加速、分布式系统与异构并行计算。

### 2. 一句话核心贡献
提出 SpecEyes 框架，将推测解码机制从“词元级别”提升至“智能体（工具调用）级别”，通过轻量级无工具小模型的快速推测与基于可分性的认知门控机制，成功打破了多模态大模型反复调用视觉工具所带来的串行延迟瓶颈，在不牺牲准确率的前提下实现了推理速度和并发吞吐量的大幅提升。

### 3. 使用指南
*   **输入**：用户的自然语言查询 (Query) 和图像输入 (Image)。
*   **输出**：模型生成的最终文本回答。
*   **工作流程 (四阶段管道)**：
    1.  **工具必要性筛选**：快速判断当前查询是否仅依赖全局图像即可回答。
    2.  **小模型推测**：使用轻量级、无工具的多模态小模型直接生成候选答案。
    3.  **认知门控判断**：计算小模型答案的“可分性分数 (Separability Score)”，若置信度高于设定阈值则直接输出该答案（跳过大模型）。
    4.  **智能体回退 (Fallback)**：若置信度过低，则将任务交由重型智能体大模型，执行完整的“感知-推理-工具调用”循环。
*   **硬件要求**：标准商用 GPU 即可运行（论文实验基于单张 NVIDIA A100 40GB GPU）。支持利用异构并发特性提升吞吐。
*   **代码开源情况**：论文未在正文中显式提供 GitHub 链接，但其方法兼容并构建于主流开源基座模型（如 Qwen-VL 系列）之上，易于复现。

### 4. 主要创新点
1. **智能体级别的推测加速 (Agentic-level Speculative Bypass)**：突破了现有工作仅在 Token 级别进行推测的局限。它允许对于不需要复杂多步视觉交互的简单查询，直接通过“系统快思考”（小模型直觉）生成答案，彻底免去繁琐的感知/推理/工具调用循环。
2. **基于答案可分性的认知门控 (Cognitive Gating via Answer Separability)**：弃用了传统容易导致过度自信的 Softmax 概率，设计了一种免校准、尺度不变的置信度指标。该指标通过量化最高 Logit 与其他竞争 Logit 之间的距离来评估预测的可靠性，无需依赖真实标签 (Oracle Labels) 即可实现精准的拦截与回退。
3. **异构并行漏斗架构 (Heterogeneous Parallel Funnel)**：在系统层面最大化硬件利用率。将小模型无状态、高并发的批处理执行 (Stateless Concurrency) 与大模型有状态、串行的工具链执行 (Stateful Serial Execution) 进行解耦，将推测接受率直接转化为系统吞吐量的乘数级增长。

### 5. 实验效果
*   **评估基准**：在涵盖细粒度感知 (V* Bench)、高分辨率理解 (HR-Bench) 和幻觉鲁棒性测试 (POPE) 三个核心多模态数据集上进行了验证。
*   **性能表现**：SpecEyes 实现了极佳的速度-精度权衡（Pareto 最优）。在大幅降低延迟并提升吞吐量的同时，不仅维持了纯智能体基线的准确率，甚至在部分任务中有所超越（例如：有效抑制了过度的工具调用导致的幻觉，在 V* 的相对位置任务中准确率从 82.89% 提升至 89.47%，在 POPE 数据集中准确率从 82.29% 提升至 83.99%）。批处理 (Batch Size) 越大，其加速和吞吐量增益的优势越显著。

## 📄 PEARL: Personalized Streaming Video Understanding Model

- **链接**: https://huggingface.co/papers/2603.20422
- **阅读来源**: HTML

### 1. 应用领域
多模态大模型-流式视频理解 / 计算机视觉-个性化视频问答

### 2. 一句话核心贡献
首次提出“个性化流式视频理解（PSVU）”任务及评测基准 PEARL-Bench，并设计了免训练、即插即用的 PEARL 框架，解决了视觉语言模型（VLM）在连续视频流中难以实时注册并推理用户自定义概念（如特定人物、物体或动作）的难题。

### 3. 使用指南
*   **输入**：连续的流式视频序列，以及用户的多轮自然语言指令（包含在任意时间戳动态定义的个性化概念，及针对该概念的实时或历史状态查询）。
*   **输出**：基于视频流上下文和注册概念生成的实时自然语言响应。
*   **部署方式**：该方法为**免训练、即插即用**架构。仅需结合现成的视觉语言模型（作为推理生成与描述提取底座，如 Qwen-VL、LLaVA-OV）和多模态嵌入模型（用于特征检索，如 Qwen3-VL-Embedding）即可运行。
*   **硬件要求**：由于涉及流式视频处理和大模型推理，需要显存充裕的 GPU 设备（论文实验采用 NVIDIA H200），具体算力需求取决于所挂载的 VLM 参数量大小。
*   **开源情况**：论文明确表示代码已开源。

### 4. 主要创新点
1.  **开创性任务与基准定义 (PSVU & PEARL-Bench)**：首次将个性化理解从静态图像/离线短视频扩展到流式长视频领域，并构建了首个包含132个长视频和2173个时间戳标注的基准测试，同时支持“帧级别（静态实体）”和“视频级别（连续动作序列）”的双层个性化概念评测。
2.  **双粒度内存系统 (Dual-grained Memory System)**：创新性地将“概念知识”与“流式观察”解耦。构建了**概念内存**（Concept Memory）用于存储用户自定义概念的视觉证据和生成的文本描述，以及**流式内存**（Streaming Memory）用于增量存档连续视频片段的紧凑多模态嵌入，大幅降低了长视频历史的维护成本。
3.  **概念感知检索算法 (Concept-aware Retrieval Algorithm)**：提出一种高效的免训练检索机制。通过大模型利用已保存的概念描述重写用户的查询（将抽象的自定义名称转换为显式的视觉/动作语义描述），从而在流式内存中精确、低延迟地检索出回答问题所需的历史视频片段证据。

### 5. 实验效果
在 PEARL-Bench 核心数据集上，PEARL 框架超越了包含开源与闭源在内的 8 种主流离线及在线视频理解模型，取得 SOTA 表现：
*   **高鲁棒性与泛化性**：作为即插即用模块应用于 3 种不同架构（LLaVA-OV-7B, Qwen2-VL-7B, Qwen3-VL-8B）时，均带来显著且一致的性能跃升，帧级别平均准确率提升 **13.79%**，视频级别平均提升 **12.80%**。
*   **超越最强基线与闭源巨头**：最高配置的 Qwen3-VL-8B+PEARL 不仅大幅超越最强的在线基线模型 StreamForest-7B（领先 **17.22%**），在极具挑战的视频级别任务上更是以近 **24%** 的巨大优势击败了强大的闭源模型 Gemini3-pro-preview。
*   **高效率**：在实现大幅精度提升的同时，由于检索和重写模块的高效设计，框架仅引入了极小的端到端延迟开销，完全满足实时流式视频交互的需求。

## 📄 ThinkJEPA: Empowering Latent World Models with Large Vision-Language Reasoning Model

- **链接**: https://huggingface.co/papers/2603.22281
- **阅读来源**: HTML

### 1. 应用领域
计算机视觉与机器人学 - 视频预测 / 潜在世界模型（Latent World Models）/ 具身智能（如3D手部操作轨迹预测）。

### 2. 一句话核心贡献
提出了一种结合大视觉语言模型（VLM）和潜在世界模型（JEPA）的双时序框架 **ThinkJEPA**，通过引入 VLM 的长视距语义指导，解决了传统密集预测模型难以捕捉长程宏观语义且容易产生局部物理偏差的问题。

### 3. 使用指南
*   **输入**：历史视频观察数据（分为两路：用于 JEPA 的短窗口密集采样帧，以及用于 VLM 的大跨度均匀采样帧），以及可选的上下文文本提示（Text Prompt）。
*   **输出**：预测的未来潜在表征（Latent tokens），该表征可通过轻量级的下游任务头（Task Head）直接解码为具体的物理预测（如未来的 3D 关节/手部运动轨迹）。
*   **硬件需求**：由于需要同时运行视频视觉大语言模型（如 Qwen-VL 作为思考器）和 V-JEPA 骨干网络，对 GPU 显存和算力有较高要求。
*   **代码开源**：文中对比了开源模型，但未在当前文本中明确声明自身代码库的开源地址（通常需关注论文最终版的 GitHub 链接）。

### 4. 主要创新点
1.  **双时序感知野采样架构 (Dual-Temporal Perception Field Sampling)**：设计了互补的双分支输入策略。JEPA 分支采用短窗口密集采样以捕捉高频细粒度的物理运动和交互线索；VLM 分支采用大步长的均匀稀疏采样，在控制计算成本的同时获取长视距的全局上下文和语义。
2.  **分层金字塔表示提取 (Hierarchical Pyramid Representation Extraction)**：摒弃了仅使用 VLM 最后一层（易丢失空间细节且过度向语言生成对齐）的做法，提取并聚合了 VLM 的视觉分词器（ViT）特征与多个中间层的隐藏状态，最大程度保留了视觉推理线索和空间敏感度。
3.  **基于 FiLM 的引导信号注入 (Complementary Guidance via FiLM)**：将 VLM 提取的多层特征映射为特征级线性调制（FiLM）参数，逐层注入到 V-JEPA 预测器中。这种方式保留了 JEPA 预测潜在物理动态的接口，同时赋予了模型跨模态的常识与高层语义引导。

### 5. 实验效果
*   **评测数据集**：在第一人称手部操作轨迹预测大规模基准 **EgoDex** 和 **EgoExo4D** 上进行了评估。
*   **下游轨迹预测表现**：全面超越了纯 VLM 基线和纯 JEPA 预测器基线。在 EgoDex 数据集上，相比最强的任务特定基线（Behavior Cloning），平均位移误差（ADE）从 0.0767 降至 0.0610，最终位移误差（FDE）从 0.0818 降至 0.0560，准确率显著提升。
*   **潜在表征预测质量**：在特征距离（FD）、潜在平滑 L1 损失（SL1）和余弦距离（CD）等纯表征层面的预测指标上同样取得了最佳效果。
*   **长程预测鲁棒性**：在长视距的递归展开（Recursive Rollout）实验中，ThinkJEPA 显著缓解了误差随时间累积的问题，生成的轨迹在时间连贯性、物理一致性和关节对齐度上表现出了极强的稳定性。

## 📄 UniGRPO: Unified Policy Optimization for Reasoning-Driven Visual Generation

- **链接**: https://huggingface.co/papers/2603.23500
- **阅读来源**: HTML

### 1. 应用领域
多模态大模型、计算机视觉-图像生成、强化学习（策略优化）

### 2. 一句话核心贡献
提出了一种统一的强化学习框架 UniGRPO，通过将“文本推理”和“视觉合成”联合建模为单一的马尔可夫决策过程（MDP），实现了对推理驱动的图文交错生成过程的联合策略优化。

### 3. 使用指南
*   **输入**：用户的文本提示（Prompt）。
*   **输出**：模型首先输出中间的推理文本（用于扩展和细化Prompt），随后输出最终合成的高质量图像。
*   **硬件要求**：需要支持大语言模型和流匹配（Flow Matching）模型联合训练的高端 GPU 集群。由于去除了训练时的分类器无分类指导（CFG），从而显著降低了训练时的显存占用和计算开销。
*   **代码开源情况**：文中未明确指出代码库链接，且基础模型（Bagel）使用了内部数据集进行监督微调（SFT），具体开源状态需等待作者团队后续发布。

### 4. 主要创新点
1.  **联合马尔可夫决策过程（MDP）建模**：将完整的“提示->文本->图像”交错生成过程公式化为一个MDP。利用稀疏的最终奖励，通过GRPO算法联合优化大语言模型的离散文本生成策略和流匹配的连续视觉生成策略。
2.  **移除CFG以实现无分支线性展开（Rollout）**：在强化学习训练阶段舍弃了计算昂贵的分类器无分类指导（CFG），确保生成过程为线性且无分支的。这不仅大幅提升了训练效率，还使其更容易扩展到多轮交互和多条件生成的复杂场景。
3.  **基于速度场MSE的鲁棒正则化**：为了防止奖励作弊（Reward Hacking），放弃了标准的潜在KL散度惩罚，转而直接在速度场（Velocity Fields）上应用未加权的均方误差（MSE）惩罚。该方法为模型提供了更直接稳健的正则化信号，有效保留了基础模型的生成先验质量。

### 5. 实验效果
在核心数据集上，UniGRPO 取得了 State-of-the-Art (SOTA) 的表现：
*   **GenEval基准（评估复杂组合能力如计数、空间关系等）**：得分为 **0.90**。
*   **内部 TA (Text Alignment) 基准**：得分为 **0.8381**。
相比于独立优化文本或图像的基线方法（如 ReFL, FPO, 单独的FlowGRPO/TextGRPO），UniGRPO 不仅实现了显著的量化性能提升，还生成了细节更逼真、指令对齐更精准的高质量图像，并展现出了更具导向性和任务相关性的思维链（CoT）推理过程。

## 📄 SIMART: Decomposing Monolithic Meshes into Sim-ready Articulated Assets via MLLM

- **链接**: https://huggingface.co/papers/2603.23386
- **阅读来源**: ArXiv Abs

### 1. 应用领域
3D计算机视觉-3D内容生成、具身AI (Embodied AI)、物理仿真

### 2. 一句话核心贡献
提出了一种统一的多模态大语言模型（MLLM）框架 SIMART，通过引入稀疏 3D VQ-VAE 显著降低 3D token 的内存开销，实现将静态的单体 3D 网格单阶段转化为可直接用于物理仿真的关节式（Articulated）3D 资产。

### 3. 使用指南

## 📄 VP-VLA: Visual Prompting as an Interface for Vision-Language-Action Models

- **阅读来源**: HTML

### 1. 应用领域
具身智能-机器人操作 (Embodied AI - Robotic Manipulation) / 视觉-语言-动作模型 (Vision-Language-Action, VLA)

### 2. 一句话核心贡献
提出 VP-VLA 双系统框架，通过结构化的视觉提示（如十字准星和边界框）作为中间接口，将高层语义推理与底层动作控制解耦，显著提升了机器人操作的空间精度和在分布外（OOD）场景下的鲁棒性。

### 3. 使用指南
*   **输入**：自然语言指令与多视角的视觉观察图像（如固定第三人称和机械臂末端第一人称视角的RGB图像）。
*   **输出**：机器人底层的连续控制信号（Action chunks，动作序列）。
*   **使用流程**：
    1.  利用物理状态变化（如夹爪开闭）作为事件触发器，调用预训练的 VLM（系统2）进行子任务分解。
    2.  VLM 输出目标对象和放置位置，调用分割模型生成带有结构化视觉提示（目标十字准星、目标区域边界框）的接口图像。

### 4. 主要创新点
1.  **解耦的双系统架构设计**：打破传统 VLA 模型将语义理解、空间定位和控制混为一体的“黑盒”映射，设计了“系统2（规划器）”与“系统1（控制器）”，并创新性地使用在图像空间叠加的“视觉提示”作为两者之间的沟通接口。
2.  **事件驱动的动态视觉提示机制**：不采用计算昂贵的逐帧推理，而是通过机器人物理交互状态的变化（如夹爪状态改变）来触发高层任务分解更新。根据不同的动作基元动态生成精确的几何先验（如抓取任务生成目标中心的十字准星，放置任务生成目标区域的边界框）。
3.  **创新的辅助视觉定位目标（Visual Grounding Objective）**：在训练阶段的关键帧中引入了基于离散化空间维度的分类定位损失（Grounding Loss），强制策略模型真正理解并利用视觉提示中的空间坐标，避免模型将其视作无关的图像噪声，有效保证了动作生成的空间对齐。

*   **Robocasa-GR1-Tabletop 仿真**：在包含多步复杂操作的基准上，平均成功率达到 53.8%，比 QwenOFT 基线提升 5.0%，在具有随机初始化和新颖外观的 OOD 任务上优势尤为明显，整体超越 GR00T-N1.6 等强基线。
*   **SimplerEnv 仿真**：取得了 58.3% 的 SOTA 平均成功率，较基线实现了 8.3% 的绝对提升。
*   **真实世界桌面操作**：在杂乱场景下的分类、特定颜色拾取、精确网格放置（蛋盒任务）等任务中，不但域内（In-Distribution）成功率极高（最高达 90%+），在未见过的物体、颜色组合及空间位置等分布外（OOD）场景下依然保持了极强的鲁棒性（例如：分类任务 OOD 成功率达 85%，远超基线的 63.3%），几乎消除了以往模型严重的泛化衰减。

## 📄 TrajLoom: Dense Future Trajectory Generation from Video

- **链接**: https://huggingface.co/papers/2603.22606
- **阅读来源**: HTML

### 1. 应用领域
计算机视觉 - 视频理解与可控视频生成（运动预测、密集轨迹生成）。

### 2. 一句话核心贡献
提出了一种名为 TrajLoom 的生成式框架，能够根据视频上下文和历史轨迹精确预测长视角（81帧）的未来密集点轨迹，显著提升了长时间运动预测的时空一致性，并可直接赋能下游的可控视频生成与编辑任务。

### 3. 使用指南
*   **输入**：过去81帧的历史视频片段、提取出的历史密集点轨迹及可见性掩码（Visibility）、可选的文本描述（Caption）。
*   **输出**：未来81帧的密集点轨迹坐标及其可见性状态。
*   **硬件需求**：模型训练计算量较大（论文中使用了多张 H100 和 RTX PRO 6000 GPU）；推理阶段支持基于 10 步 Euler 的轻量级 ODE 采样，常规支持深度学习推理的商用 GPU 即可运行。
*   **开源情况**：作者已明确开源完整的代码（Code）、模型权重（Checkpoints）以及统一评测基准数据集（TrajLoomBench）。

### 4. 主要创新点
1.  **网格锚点偏移编码 (Grid-Anchor Offset Encoding)**：摒弃了使用绝对坐标的方式，将每个轨迹点表示为相对于固定像素中心锚点的偏移量。该机制有效消除了由空间网格位置引入的统计偏差，使模型能够专注于局部运动特征，为潜在生成模型提供了稳定的基础。
2.  **时空一致性正则化的轨迹自编码器 (TrajLoom-VAE)**：设计了一种变分自编码器将高维密集轨迹压缩为紧凑的时空 Token。除了掩码重构损失外，引入了专门的**时空一致性正则化器**，通过对齐时间帧上的速度和空间上的多尺度邻居关系，有效消除了轨迹重构中的时间抖动和局部空间撕裂。
3.  **基于流匹配的条件轨迹生成 (TrajLoom-Flow)**：在 VAE 的潜在空间中，使用校正流（Rectified Flow）条件化地生成未来轨迹。为解决长程预测的漂移问题，引入了**边界提示（Boundary Cues）**来强制未来预测与历史观测连续，并提出了 **On-policy K步微调**策略来缓解训练与推理阶段 ODE 采样的分布不匹配问题。

### 5. 实验效果
在涵盖真实和合成视频的统一基准 **TrajLoomBench**（包含 Kinetics, RoboTAP, Kubric, MagicData）上，TrajLoom 取得了 SOTA（State-of-the-Art）表现：
*   **预测长度翻倍**：将高质量未来轨迹的预测窗口从先前的 24 帧大幅扩展至 **81 帧**。
*   **运动逼真度与稳定性大幅提升**：相比于目前的最佳方法 WHN (L)，TrajLoom 显著降低了特征级视频运动距离（如在 Kubric 数据集上 FVMD 从 4872 骤降至 1338），同时在物理流场指标（FlowTV 和 DivCurlE）上表现更优，表明其生成的运动具有更少的空间不连续性和异常旋转/形变。
*   **下游任务直接可用**：预测出的长时间密集轨迹可以直接作为即插即用的控制信号，输入给 Wan-Move 等视频扩散模型，成功实现高度运动一致的可控视频合成。

*   **使用方式**：PEPO 是一个即插即用的策略优化模块，可无缝集成到现有的 GRPO 或 DAPO 等基于可验证奖励的强化学习框架中。只需在训练的优势函数（Advantage）计算阶段，提取模型的隐藏层状态和 Logits 即可进行 Token 权重重分配。
*   **硬件与效率**：不需要任何特殊的硬件或辅助网络分支。实验表明，其引入的单步额外计算开销小于 1%，整体训练吞吐量与标准 GRPO 基本持平（在 8 张 NVIDIA A40 GPU 上验证）。
*   **代码基础**：论文基于开源的 Swift 框架实现，支持梯度检查点（Gradient Checkpointing）和 bfloat16 混合精度训练。

### 4. 主要创新点
1. **细粒度的感知-探索互补机制**：打破了传统 GRPO 序列级奖励均匀分配的局限，首次指出多模态成功推理依赖于少数“视觉锚定 Token（感知）”与“高熵过渡 Token（探索）”的互补，实现了更科学的 Token 级信用分配。
2. **无需额外监督的视觉感知先验提取**：直接计算响应 Token 与视觉 Token 在各 Transformer 层隐藏状态的余弦相似度，作为 Token 的视觉依赖得分。该方法无需引入复杂的掩码（Masking）或辅助评估网络，轻量且高效。
3. **平滑门控融合机制（Smooth Gating Mechanism）**：设计了一种归一化并均值居中的平滑门控算子，将视觉相似度先验与基于输出 Logits 的信息熵进行动态融合。该机制既能突出视觉相关的推理步骤，又能引导模型在不确定节点进行探索，同时保持全局策略梯度的尺度稳定。

### 5. 实验效果
在 Qwen2.5-VL-3B-Instruct 和 InternVL3-2B 等模型上进行了广泛评估，PEPO 表现出远超基线（GRPO、DAPO、仅高熵 RL）的性能和训练稳定性：

## 📄 Uncertainty-guided Compositional Alignment with Part-to-Whole Semantic Representativeness in Hyperbolic Vision-Language Models

- **链接**: https://huggingface.co/papers/2603.22042
- **阅读来源**: HTML

### 1. 应用领域

### 2. 一句话核心贡献
提出了一种名为 UNCHA（不确定性引导的组合双曲对齐）的视觉语言模型，通过在双曲空间中利用“不确定性”来量化图像局部对整体的语义代表性差异，解决了现有模型在复杂多目标场景下难以有效捕捉层级和组合结构的问题。

### 3. 使用指南
* **输入**：图像（包括整体场景图及局部裁剪图）与文本描述（包括整体描述及局部对象描述）。
* **输出**：保留了层级与组合关系的多模态特征向量（嵌入在双曲空间中），可直接计算双曲距离（如洛伦兹内积）用于分类、检索或分割任务。
* **硬件需求**：预训练阶段需要较高算力支持（论文使用 4 张 A100 GPU 进行 500K 步训练）；推理阶段硬件需求与常规的基于 ViT-S/ViT-B 的 CLIP 模型相当。

1. **基于双曲不确定性的语义代表性建模**：首次利用双曲半径（通过欧式范数近似计算以保持数值稳定）作为不确定性的代理指标。对整体场景代表性强的局部图像分配较低的不确定性，对代表性弱（如模糊、无纹理）的局部图像分配较高的不确定性。
2. **不确定性引导的对比损失 (Uncertainty-guided Contrastive Loss)**：摒弃了传统对比学习中固定的温度系数，引入了与局部不确定性自适应挂钩的动态温度参数。不确定性越高的局部特征具有越大的温度值，从而降低其在全局-局部对比损失中的权重，避免强制对齐带来的语义结构破坏。
3. **结合熵正则化的蕴含损失校准 (Entailment Loss with Uncertainty Calibration)**：在捕捉“局部-整体”层级关系的蕴含损失中引入了不确定性校准模块，并设计了基于熵的正则化项（Entropy-based regularizer）。该机制能够防止模型将特征过度集中在双曲空间的狭窄区域（避免空间塌陷），促使特征根据抽象程度在双曲空间中得到更均匀、高效的分布。

### 5. 实验效果
该方法在多个核心数据集上实现了 State-of-the-Art (SOTA) 的表现：
* **零样本分类与检索**：在 16 个零样本图像分类基准（如 ImageNet 等）和 COCO 文本-图像双向检索任务中，持续超越现有的欧式模型（如 CLIP）和双曲模型（如 ATMG, HyCoCLIP）。
* **多目标组合理解**：在 COCO 多标签分类任务，以及 ComCo 和 SimCo 这类专门评估多目标/组合理解能力的数据集上，平均精度（mAP）获得显著提升，证明了模型对复杂场景中多物体的解耦与对齐能力极强。
* **细粒度与层级对齐**：在带困难负样本的局部特征对齐基准（DCI）和零样本语义分割上表现突出，嵌入空间可视化分析证实，UNCHA 能清晰地实现“局部靠近原点，整体远离原点”的严格层级排序。

*   **输入**：复杂推理任务的提示词（如数学题或逻辑题）。
*   **输出**：模型生成的逐步推理轨迹（Chain-of-Thought）及最终答案。
*   **使用方法**：
    1.  **分析诊断（Cross-Sampling 交叉采样）**：在推理生成时，设定 Token 级别的 JS 散度阈值。在大多数位置使用 Base 模型生成，仅在高散度位置替换为 RL 微调模型的 Token，以此低成本验证和恢复 RL 模型的强大推理能力。
    2.  **训练优化（Divergence-Weighted Advantages）**：在采用 GRPO 或 DAPO 等 RLVR 训练算法时，修改目标函数，将每个 Token 的 Advantage（优势信号）乘以该 Token 相对旧策略的 KL 散度权重（通过 Sigmoid 函数映射），以强化或平滑特定 Token 的学习信号。
*   **硬件需求**：常规的大模型推理/微调硬件（如 A100/H100 GPU 集群），兼容 `verl` 或 DeepSpeed 框架。
*   **代码开源情况**：论文实验基于开源的 DAPO 训练配方与 `verl` 框架构建，所提方法（如散度加权、交叉采样）仅需在开源 RLHF 框架推理和计算 Advantage 阶段进行轻量级代码修改即可实现。

### 4. 主要创新点
1.  **提出基于 Token 级别的稀疏分布偏移视角**：系统性量化了 Base 模型与 RL 模型之间的 Token 预测差异，发现大多数位置的 JS 散度几乎为零，RLVR 的修改极其稀疏且多集中在序列的早期分支决策和末尾格式处。
2.  **设计前向与反向交叉采样（Cross-Sampling）干预机制**：首次通过动态融合两种策略（Base与RL）的干预实验，在因果层面上证明了：仅需极小比例（高散度）的 Token 改变，就足以决定整个序列推理的成败。
3.  **揭示 RLVR 内部“重排”机制并提出散度加权目标（Divergence-Weighted Advantages）**：发现 RL 提升性能本质上是提升 Base 模型中已有合理候选 Token 的优先级（而非无中生有）。基于此发现，设计了根据 Token 级别散度自适应调整 Advantage 权重的训练干预方法，为未来设计更精细的 RL 目标函数提供了新范式。

### 5. 实验效果
在 AIME 2024、AIME 2025、AMC 和 GPQA 等高难度基准测试上（基于 Qwen2.5-32B/7B 等模型），取得了以下关键结果：
*   **极高的干预效率（功能验证）**：在基础模型生成轨迹时，只需替换平均 **1% - 10%** 的关键 RL Token（前向交叉采样），即可将基础模型的准确率提升至与 RL 完整模型相当（例如从约 20% 跃升至 50% 以上），甚至在某些情况下超越单纯的 RL 模型。反之，替换少量 RL 模型的 Token 为基础模型 Token 会导致性能迅速崩溃。
*   **训练性能提升**：将理论发现转化为训练策略（散度加权优势），在 Qwen2.5-Math-7B 上，相比标准 DAPO/GRPO 基线，采用低 KL 或高 KL 增强权重的策略均能在 AIME 2024/2025 和 AMC 测试集上带来进一步的准确率提升。
*   **相较 SFT 的优势**：对比实验表明，RLVR 实现的推理能力跃升比监督微调（SFT）所需修改的 Token 数量和参数权重分布少得多，证明 RL 是一种高度精准的“手术刀式”优化，而非 SFT 那样的全局重塑。

* **词汇特征**：PCA降维和词频分析证实，具有RLHF对齐训练的模型，无论参数大小，均高度聚集于富含“权利、尊严、公平”等高阶道德词汇的空间中，证实对齐训练主导了修辞风格。

## 📄 From Static Templates to Dynamic Runtime Graphs: A Survey of Workflow Optimization for LLM Agents

- **链接**: https://huggingface.co/papers/2603.22386
- **阅读来源**: HTML

### 1. 应用领域
自然语言处理（NLP） - 大模型智能体（LLM Agents）、多智能体系统（Multi-Agent Systems）与工作流优化（Workflow Optimization）。

本文将LLM智能体系统统一抽象为“智能体计算图（ACGs）”，系统性综述了从静态模板到动态运行时图的工作流优化方法，并提出了包含图层级属性和成本考量的结构感知标准化评估协议。

### 3. 使用指南
*   **适用对象**：从事大模型智能体开发、复杂任务流编排与多智能体系统优化的研究人员及系统工程师。
*   **核心资源**：作者开源了文献汇总与分类仓库（GitHub: `https://github.com/IBM/awesome-agentic-workflow-optimization`），可作为查阅最新agent框架和优化算法的索引。
*   **方法范式概览**：
    *   **输入**：任务指令、基础LLM、工具注册表（Tool Registry）、验证器（Verifiers）等。
    *   **输出**：优化的可执行工作流（分为可复用的**静态模板**，或针对特定输入动态生成/编辑的**运行时计算图**）。
    *   **优化信号**：使用标量指标（如准确率）、验证器信号（如代码单元测试）、偏好数据（Preference）或执行轨迹衍生的文本反馈（Trace-derived feedback）来指导图结构的更新。
*   **硬件与运行要求**：无特定硬件要求，具体算力需求取决于所研究的特定智能体框架（如 DSPy, AutoGen）及工作流的复杂度。

### 4. 主要创新点
1. **提出统一的理论抽象（Agentic Computation Graphs, ACGs）**：将文献中散乱的“Pipelines”、“协同框架”、“通信拓扑”等概念统一为包含节点（LLM调用、工具使用、验证）和边（控制流、信息流）的计算图，确立了“工作流结构”本身作为第一级优化目标的地位。
2. **构建基于图可塑性的多维分类法（Taxonomy）**：创新性地基于“结构决定时机”对现有研究进行分类，区分了**静态模板优化**（离线约束空间搜索）与**动态方法**（包含超图选择与剪枝、执行前结构生成、执行中交错编辑），并引入图决定时间（GDT）和图可塑性模式（GPM）来精确描述中间状态的方法。
3. **提出结构感知的评估协议（Structure-aware Evaluation Protocol）**：指出当前领域仅报告“下游任务成功率”的弊端，提出了一套包含计算图属性（节点数、深度、通信量）、执行成本（Token、API调用次数、延迟）、结构鲁棒性和控制流隐藏行为的“最小报告清单”，以提升工作流优化的科学性和可比性。

### 5. 实验效果
作为一篇系统性综述，本文未提出单一新算法进行跑分，而是综合评估了77篇核心前沿文献的实验结论，提炼出以下具有指导意义的实证规律（Empirical Patterns）：
*   **静态与动态的边界**：在算子空间受限、评估信号强且任务模式重复时（如具备单元测试的代码生成），**离线静态搜索**（Static Search）能以更低成本击败动态生成；但当面临工具漂移或极强输入异构性时，静态模板容易崩溃。
*   **动态优化的成本效益**：当任务变化仅体现在难度上时，**选择与剪枝（Selection/Pruning）**最具性价比；当任务结构本质不同时，需采用**执行前生成（Pre-execution generation）**；仅当环境存在不可预知的强交互反馈时，才必须使用成本最高昂的**执行中编辑（In-execution editing）**。
*   **结构优化与提示词微调的对比**：综合文献表明，当智能体的失败源于缺乏中间验证、任务分解错误或通信冗余时，进行图级别的拓扑优化（Graph Optimization）带来的性能飞跃显著大于单纯的提示词微调（Prompt Tuning）。

## 📄 MultiBind: A Benchmark for Attribute Misbinding in Multi-Subject Generation

- **链接**: https://huggingface.co/papers/2603.21937
- **阅读来源**: HTML

### 1. 应用领域
计算机视觉 - 文本到图像生成（具体为：多主体、多参考驱动的图像定制与编辑）

### 2. 一句话核心贡献
提出了 **MultiBind** 基准及一种分维度的混淆评估协议，首次系统性地量化并诊断了多主体图像生成中长期被忽视的“属性错误绑定”问题（如特征漂移、身份交换、属性混合）。

### 3. 使用指南
*   **输入（针对生成模型）**：多张独立主体（Subject）的参考图像、一张去除了主体的背景参考图像，以及一段结构化的、带有实体索引的长文本提示词（如明确指定“Subject A穿什么，Subject B做什么”）。
*   **输出（针对生成模型）**：一张将所有主体融合在同一背景下，并遵循提示词属性编辑的多人场景合成图像。
*   **评测流程**：评估管线接收生成的图像，通过目标检测/分割提取生成的主体，利用特定维度的专家模型（人脸、外观、姿态、表情）计算相似度矩阵，并输出多维度的失败模式诊断报告（包括成功率、漂移率、混合率、交换率等）。
*   **硬件要求**：运行评估管线仅需常规GPU以加载各维度的特征提取模型（如ArcFace、ViTPose等）和基础匹配算法，无需极高算力。
*   **开源情况**：论文指出数据集注释（掩码、边界框）、规范化的图像资源、结构化提示词构建规则以及评估诊断代码均包含在配套文件中，面向社区开源。

### 4. 主要创新点
1.  **基于真实图像锚定（Real-image Grounding）的基准构建**：区别于使用合成数据，MultiBind 基于真实的多人照片构建。它提供精确的槽位级别（Slot-level）监督信号（实例掩码、边界框），并通过严格的管道生成规范化的主体参考和背景，确保了复杂提示词的内部一致性与评估的确定性。
2.  **分维度的专家评估协议（Dimension-wise Specialist Protocol）**：摒弃了粗糙的全局CLIP相似度，引入了专门针对面部身份（Face）、外观（Appearance）、姿态（Pose）和表情（Expression）的专家模型，实现了特征解耦的细粒度评估。
3.  **基线校准的混淆矩阵诊断（Baseline-corrected Confusion Diagnostics）**：通过计算生成主体与真实目标间的相似度矩阵，并**减去**真实主体之间固有的相似度基线，有效剥离了模型单纯的“生成质量退化（Drift）”与真正的“跨主体特征干扰（如Swap, Dominance, Blending）”，使抽象的生成错误具象化、可解释。

### 5. 实验效果
在包含真实多人场景的测试集上，作者评估了6种先进的多主体生成模型（闭源：Nano Banana Pro, GPT-Image-1.5, Seedream 4.5；开源：HunyuanImage-3.0, Qwen-Image-Edit, OmniGen2），主要表现如下：
*   **揭示隐藏缺陷**：传统的全局指标（FID, CLIP等）无法反映属性绑定错误。例如，Seedream 4.5 和 Hunyuan 在全局指标上表现接近，但 MultiBind 揭示出 Seedream 极易发生**面部特征混合（Blending）**，而 Hunyuan 主要表现为**特征退化/丢失（Drift）**。
*   **模型阵营对比**：闭源模型（尤其是 Nano Banana Pro 和 GPT-Image-1.5）在成功率和稳定性上全面领先，属性绑定最精准。开源模型（如 Qwen 和 OmniGen2）极不稳定，在外观和表情维度上出现了高达近20%的**属性交换（Swap）**和严重的**主导覆盖（Dominance）**现象。
*   **人类对齐率**：经过基于AUC的元评估验证，该方法提供的配对级混淆分数（Delta scores）在与人类判断的一致性上，显著超越了直接使用强视觉大模型（如Gemini 2.5 Pro或GPT-4o级模型）作为裁判的基线方案。

## 📄 MinerU-Diffusion: Rethinking Document OCR as Inverse Rendering via Diffusion Decoding

- **链接**: https://huggingface.co/papers/2603.22458
- **阅读来源**: HTML

### 1. 应用领域
计算机视觉 - 文档OCR（光学字符识别与结构化文档解析）

### 2. 一句话核心贡献
提出 MinerU-Diffusion 框架，将文档OCR任务从传统的自回归顺序生成重新定义为基于块状扩散解码的视觉条件逆渲染过程，通过并行去噪大幅提升了长文档解析的推理速度并有效缓解了错误累积与语义幻觉问题。

### 3. 使用指南
*   **输入**：二维文档图像（支持完整PDF页面、公式截屏、表格截屏等，支持原生分辨率及图像自适应缩放）。
*   **输出**：包含结构化排版信息的1D Token序列（统一输出文本、布局坐标、Markdown/LaTeX公式、OTSL表格标签等）。
*   **硬件需求**：模型参数量为 2.5B（基于Qwen2-VL和SDAR等初始化），推理端支持KV-Cache复用以加速计算，论文在单张 NVIDIA H200 GPU 上进行了吞吐量（TPS）测试评估。
*   **使用方式**：支持通过动态置信度阈值（Dynamic confidence threshold）控制并行度，在推理时可根据场景需求在“高吞吐量”与“高准确率”之间灵活权衡。
*   **开源情况**：模型权重已在 Hugging Face 开源（地址：`opendatalab/MinerU-Diffusion-V1-0320-2.5B`）。

### 4. 主要创新点
1.  **块状注意力扩散解码架构 (Block-wise Diffusion Decoding)**：摒弃了计算复杂度随序列长度呈二次增长的全注意力机制，采用“块内并行扩散、块间自回归”的混合注意力掩码结构。这一设计在实现局部并行生成的同时，利用块边界作为结构锚点，防止了长程位置漂移，实现了内存和计算成本的近线性扩展。
2.  **不确定性驱动的两阶段课程学习 (Two-Stage Curriculum Learning)**：为解决扩散模型在任意顺序建模中的训练不稳定性，设计了两阶段训练法：第一阶段在大规模、高熵数据上建立基础鲁棒表征；第二阶段基于模型生成结果的“不确定性评分”筛选出长尾困难样本（如复杂排版、密集公式），进行边界精细化微调，打破了性能瓶颈。
3.  **降低语言先验依赖的视觉还原机制**：传统自回归模型在视觉信号微弱时极易依赖大模型的语言先验产生“幻觉”。本文构建了纯视觉对应的扩散解码方案，并在新提出的 Semantic Shuffle（语义打乱）基准上证明：当文本语义逻辑被破坏时，MinerU-Diffusion 依然能忠实于视觉输入进行准确重构，展现出远超自回归框架的抗干扰鲁棒性。

### 5. 实验效果
*   **核心数据集表现**：在 OmniDocBench v1.5（全页面解析）、UniMER-Test（公式识别）和 CC-OCR/OCRBench v2（表格识别）等核心基准测试中，整体解析精度（如 Overall 93.37）均达到与当前最先进的自回归基线模型（如 MinerU 2.5）相媲美或更优的水平。
*   **解码效率**：相较于传统的逐字生成基线，在维持 99.9% 相对准确率时速度提升 2.12 倍；在维持 98.8% 相对准确率时（动态阈值设为0.6），解码速度（吞吐量）最高提升了约 3.2 倍。
*   **鲁棒性表现**：在多尺度、复杂排版以及手写体/印刷体混合的 OCR 场景下表现稳定，有效解决了长序列生成中的重复和早期错误级联传播问题。

## 📄 2Xplat: Two Experts Are Better Than One Generalist

- **链接**: https://huggingface.co/papers/2603.21064
- **阅读来源**: HTML

### 1. 应用领域
计算机视觉 - 3D重建与新视角合成（3D Reconstruction and Novel View Synthesis / 3D Gaussian Splatting）

### 2. 一句话核心贡献
提出了一种基于“双专家”（几何专家与外观专家解耦）架构的**无位姿前向 3D 高斯溅射（3DGS）框架**，打破了传统单一庞大网络的性能瓶颈，实现了从无标定多视角图像中高效、高保真地生成 3D 场景表示。

### 3. 使用指南
*   **输入**：任意数量的未标定多视角图像（无需提供 Ground-Truth 相机位姿或内参）。
*   **输出**：像素对齐的 3D 高斯表示（3D Gaussians）以及预测的相机参数，可直接用于高质量的新视角图像渲染。
*   **硬件需求**：
    *   **训练/微调**：由于使用了大容量预训练模型，训练阶段需要较高的算力（论文使用了 8 张 H200 GPU）。
    *   **推理**：计算效率高，在单张消费级显卡（如 RTX 3090）上即可完成端到端的快速推理。
*   **开源情况**：论文已提供 Project page，通常意味着相关代码和预训练权重会开源，用户可直接加载预训练的“双专家”权重进行轻量级微调或直接推理。

### 4. 主要创新点
1.  **显式解耦的“双专家”模块化架构 (Two-Expert Architecture)**：摒弃了主流的“大一统”单网络联合预测范式，将任务拆分为专门的“几何专家”（负责预测相机位姿，如 DA3）和“外观专家”（负责生成 3DGS，如 MVP），有效避免了几何推理与外观建模在共享特征表示下的相互干扰。
2.  **位姿条件注入的高级外观建模 (Pose-Conditioned Appearance Modeling)**：通过将几何专家预测出的相机位姿显式传递给外观专家，使得网络能够充分利用先进的位姿感知架构机制（如极线Transformer），从而降低了网络从零学习几何结构的负担，大幅提升了生成 3DGS 的保真度。
3.  **极高效率的端到端联合优化 (Efficient End-to-End Joint Optimization)**：框架直接复用了两个成熟的预训练专家模型，无需引入从头开始初始化的新模块。这不仅使得外观专家通过联合优化对带有噪声的相机位姿具有鲁棒性，还让整个模型在不到 5000 次的迭代内即可收敛，训练效率远超现有方法。

### 5. 实验效果
*   **核心数据集表现**：在 DL3DV 和 RE10K 数据集上，该方法大幅超越了此前最先进的无位姿前向 3DGS 方法（如 YoNoSplat），且性能达到了需要真实相机位姿（Posed）的 SOTA 方法的同等水平。
*   **鲁棒性与扩展性**：随着输入视角数量的增加（从 6 视角扩展至最高 128 视角），模型的渲染质量持续提升，而基线方法在此情况下通常会出现性能衰减。
*   **泛化能力**：在跨数据集零样本泛化测试中（在 DL3DV 上训练，在 ScanNet++、Tanks&Templates 和 Mip-NeRF360 上直接测试），仍保持了极强的渲染质量和多视角一致性。
*   **训练效率**：相较于基线方法 YoNoSplat 需要 16 张 GPU 训练 150K 次迭代，本文方法仅需 8 张 GPU 训练不到 5K 次迭代，训练成本呈数量级下降。

## 📄 VISion On Request: Enhanced VLLM efficiency with sparse, dynamically selected, vision-language interactions

- **链接**: https://huggingface.co/papers/2603.23495
- **阅读来源**: HTML

### 1. 应用领域
多模态大模型（Large Vision-Language Models, LVLMs）推理加速与效能优化。

### 2. 一句话核心贡献
提出了一种名为 **VISOR (VISion On Request)** 的方法，通过在模型中策略性地稀疏化交叉注意力层和动态分配自注意力层，替代传统的视觉Token压缩策略，在不丢失高分辨率细粒度视觉信息的前提下，大幅降低多模态大模型的推理计算成本。

### 3. 使用指南
*   **输入**：高分辨率图像与文本提示词（Query/Prompt）。
*   **输出**：文本回答（与标准多模态模型一致）。
*   **运行机制**：在推理阶段，模型内部的一个轻量级路由模块（Router）会根据当前输入样本的复杂度，动态决定需要执行多少层视觉“自注意力”层。对于简单任务只需极少计算，对于复杂任务（如文档VQA）则自动分配更多计算以微调视觉特征。
*   **硬件与部署**：不需要特殊的定制硬件，兼容主流GPU（论文在AMD MI300X上进行了测试）；兼容现有的高效注意力算子（如Flash Attention）；可以即插即用现有的标准LVLM架构（如LLaVA-OV）。
*   **正交兼容性**：该方法与现有的Token裁剪/合并（Token Reduction）方法正交，可结合使用（即VISOR-TR）以获得更极端的压缩比。

### 4. 主要创新点
1.  **解耦与稀疏的视觉-语言交互架构**：打破传统LVLM每一层都进行全量多模态拼接自注意力的范式。VISOR让绝大多数大语言模型（LLM）层仅处理文本Token；利用计算成本极低的交叉注意力（Cross-Attention）为文本提供视觉上下文；仅在少数关键层执行全量的自注意力（Self-Attention）以更新和精炼视觉特征。
2.  **支持自适应计算的统一模型训练（Universal Model）**：提出了一种子网络随机采样训练策略，通过在训练时随机丢弃不同组合的自注意力层，训练出单一的“通用模型”。该模型不仅无需针对不同计算预算保存多个权重，还能在不同的计算开销配置下保持鲁棒性，起到正则化提升泛化能力的作用。
3.  **基于离线伪标签的样本级动态路由机制**：引入了一个轻量级的策略网络（路由Token与MLP层），通过离线提取每种配置下的损失和准确率来生成“伪标签”，训练该路由网络。这使得模型在推理时能实现“样本级别（per-sample）”的自适应算力分配，杜绝了“一刀切”的算力浪费。

### 5. 实验效果
*   **任务分类对比**：在多种视觉语言基准测试中，将任务分为“简单”（如SQA、POPE，需要粗粒度理解）和“困难”（如DocVQA、ChartQA、InfoVQA，需要高分辨率和细粒度理解）。
*   **性能表现**：在“简单”任务上，匹配或超越了现有SOTA方法；在“困难”任务上，由于保留了完整的视觉上下文，彻底解决了以往Token压缩方法（如VisionZip, VisPruner）带来的严重性能骤降问题，大幅领先基线。
*   **效率提升**：相较于原始的 LLaVA-OV (0.5B/1.5B) 模型，VISOR在保持或提升整体准确率的同时，实现了高达 **70% 左右的 LLM 序列处理 FLOPs 节省**。结合简单的Token合并策略（VISOR-TR）后，计算开销可进一步极致压缩，实现了最佳的精度-效率权衡（Accuracy-Efficiency Trade-off）。
